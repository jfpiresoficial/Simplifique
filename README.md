# 📊 Monitor Simplifique

Sistema web simples para monitoramento de uso interno, desenvolvido em HTML puro para facilitar acompanhamento de acessos e utilização dentro do ambiente corporativo.

---

## 🚀 Objetivo

O projeto foi criado para auxiliar no controle e monitoramento de utilização do sistema Simplifique, permitindo visualizar informações importantes de forma rápida, prática e organizada.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📂 Estrutura do Projeto

```bash
monitor-simplifique/
│
├── monitor-simplifique.html
├── README.md
````

---

## ▶️ Como Executar

### 1. Baixe o arquivo

Certifique-se de possuir o arquivo:

```bash
monitor-simplifique.html
```

---

### 2. Abra no navegador

Basta clicar duas vezes no arquivo HTML ou abrir manualmente utilizando:

* Google Chrome
* Microsoft Edge
* Opera GX
* Firefox

## 🚀 Publicando na Vercel

### Passo 1 — Criar conta

Acesse:

[https://vercel.com](https://vercel.com)

Faça login com:

* Google
* GitHub
* Microsoft

---

### Passo 2 — Enviar projeto

Crie uma pasta chamada:

```bash
monitor-simplifique
```

Coloque dentro dela:

```bash
monitor-simplifique.html
```

---

### Passo 3 — Subir para o GitHub

Instale o Git:
[https://git-scm.com](https://git-scm.com)

Depois execute:

```bash
git init
git add .
git commit -m "Primeiro commit"
```

Crie um repositório no GitHub e conecte:

```bash
git remote add origin URL_DO_REPOSITORIO
git branch -M main
git push -u origin main
```

---

### Passo 4 — Importar na Vercel

Na Vercel:

1. Clique em **Add New**
2. Depois em **Project**
3. Selecione o repositório
4. Clique em **Deploy**
